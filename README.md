# HamidoAI GitHub Pages

## Vor dem Hochladen
Ersetze in `imprint.html` und `privacy.html` die Platzhalter durch eine vollständige ladungsfähige Anschrift.

## GitHub Pages veröffentlichen
1. Neues öffentliches Repository erstellen, z. B. `hamidoai-site`.
2. Alle Dateien hochladen.
3. `Settings` → `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`, Folder: `/ (root)`.
6. Speichern und einige Minuten warten.

Typische Links:
- Website: `https://DEIN-GITHUB-NAME.github.io/hamidoai-site/`
- Terms: `https://DEIN-GITHUB-NAME.github.io/hamidoai-site/terms.html`
- Privacy: `https://DEIN-GITHUB-NAME.github.io/hamidoai-site/privacy.html`

Hinweis: technische Vorlage, keine Rechtsberatung.
